# bloomfield

Acupuncture practice website — showcase + 3rd-party booking redirect.

## Stack
- React + Vite + TypeScript + Tailwind CSS v4
- UI: shadcn/ui + Lucide icons + Motion animations + Inter font
- Routing: React Router
- Runtime/Package Manager: Bun

## Commands
- Dev: `bun dev`
- Build: `bun run build`
- Lint: `bun run lint`
- Preview: `bun run preview`
- Add shadcn component: `bunx shadcn@latest add <component>`
